继承 Inheritance

继承的基本语法 Basic syntax of inheritance

  1. The subclass will automatically have the attributes and methods defined in the superclass.

  2. The superclass is also called the parent class or base class.

  3. The subclass is also called the derived class.

继承给编程带来的便利 The Benefits That Inheritance Brings to Programming

  1. Improves code reusability.

  2. Enhances code extensibility and maintainability.

继承的深入讨 In-Depth Discussion of Inheritance

  1. A subclass inherits all the attributes and methods of its superclass. Non-private attributes and methods can be accessed directly in the subclass, but private attributes and methods cannot; to use those, the subclass must go through public (or otherwise accessible) methods provided by the superclass.
    子类继承了所有的属性和方法，非私有的属性和方法可以在子类直接访问, 但是私有属性和方法不能在子类直接访问，要通过父类提供公共的方法去访问
  2. A subclass must invoke the superclass’s constructor to complete the initialization of the superclass.
    子类必须调用父类的构造器， 完成父类的初始化
  3. When creating an instance of a subclass, no matter which constructor of the subclass is used, by default the no-argument constructor of the superclass will be called. If the superclass does not provide a no-argument constructor, then the subclass’s constructor must explicitly use super(…) to specify which superclass constructor to invoke for initializing the superclass; otherwise, the code will fail to compile. (Illustrative examples may help clarify this.)
    当创建子类对象时，不管使用子类的哪个构造器，默认情况下总会去调用父类的无参构造器，如果父类没有提供无参构造器，则必须在子类的构造器中用 super 去指定使用父类的哪个构造器完成对父类的初始化工作，否则，编译不会通过
  4. If you wish to call a particular constructor of the superclass, you must explicitly invoke it with super(parameterList).
    如果希望指定去调用父类的某个构造器，则显式的调用一下 : super(参数列表)
  5. When using super, it must appear as the very first statement in the subclass constructor (and super can only be used inside a constructor).
    super 在使用时，必须放在构造器第一行(super 只能在构造器中使用)
  6. Both super() and this() calls must be placed as the first statement in a constructor; therefore, they cannot both appear in the same constructor.
    super() 和 this() 都只能放在构造器第一行，因此这两个方法不能共存在一个构造器
  7. In Java, every class is ultimately a subclass of Object; Object is the root (base) class of all classes.
    java 所有类都是 Object 类的子类, Object 是所有类的基类.
  8. The chain of superclass constructor calls is not limited to the direct superclass—it traces upward all the way to Object (the top-level superclass).
    父类构造器的调用不限于直接父类！将一直往上追溯直到 Object 类(顶级父类)
  9. A subclass can directly inherit from at most one superclass (i.e., Java uses single inheritance). Consider: how could class A inherit from both B and C? The solution is typically to have A extends B and then B extends C, forming a chain rather than multiple direct superclasses.
    子类最多只能继承一个父类(指直接继承)，即 java 中是单继承机制。
    思考：如何让 A 类继承 B 类和 C 类？ 【A 继承 B， B 继承 C】
  10. Do not abuse inheritance: the relationship between subclass and superclass must satisfy the “is-a” logical relationship.
    不能滥用继承，子类和父类之间必须满足 is-a 的逻辑关系

继承的本质分析 Analysis of the nature of inheritance

在 new 了一个 Son 以后，会先加载父类从 Obj 开始，依次加载 GrandPa,Father,Son 并且堆中会保存各类的属性
After creating a new Son, the parent class will be loaded first, starting from Obj, and then GrandPa, Father, and Son will be loaded in sequence, and the heap will save various attributes.

Super 关键字

  1. 访问父类的属性，但不能访问父类的私有属性 super.属性名
    Accessing superclass attributes, but you cannot access the superclass’s private attributes.
  2. 访问父类的方法，不能访问父类的私有方法 super.方法名（参数列表）
    Accessing superclass methods, but you cannot access the superclass’s private methods.
  3. 访问父类的构造器 super(参数列表)：只能放在构造器的第一句，只能出现一句
    Accessing the superclass constructor: super(parameterList);It must be placed as the first statement in the subclass constructor, and can appear only once.

super 给编程带来的便利

  1. 调用父类的构造器的好处（分工明确，父类属性由父类初始化，子类的属性由子类初始化）
    Benefits of calling the superclass constructor (clear separation of responsibilities: superclass attributes are initialized by the superclass, subclass attributes are initialized by the subclass).
  2. 当子类中有和父类中的成员（属性和方法）重名时，为了访问父类的成员，必须通过 super. 如果没有重名，使用 super、this、直接访问是一样的效果
    When a subclass has members (fields or methods) with the same names as those in the superclass, to access the superclass member you must use super. If there is no name conflict, using super, this, or direct access yields the same effect! [Example]
  3. super 的访问不限于直接父类，如果爷爷类和本类中有同名的成员，也可以使用 super 去访问爷爷类的成员；如果多个基类（上级类）中都有同名的成员，使用 super 访问遵循就近原则。A->B->C，当然也需要遵守访问权限的相关规则
    Access via super is not limited to the direct superclass. If the grandparent class and the current class both have a member with the same name, you can still use super to access the grandparent’s member; if multiple ancestor classes have members with the same name, using super follows the principle of closest match in the inheritance chain (e.g., A->B->C). Of course, you must still obey the usual access-control rules.
