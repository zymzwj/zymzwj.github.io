    1. Purpose of Routing

    In Laravel, routing defines:

    When a user visits a certain URL, what content should be returned.

    For example:

    • User visits /hello → returns the text “Hello Laravel”
    • User visits /about → returns an HTML page

    Laravel first matches the route, then hands it over to the controller or directly returns the content.
